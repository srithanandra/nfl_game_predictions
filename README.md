
# **NFL GAME WINNER PREDICTION PYTHON MODEL**

IN ORDER TO RUN:
- Clone the repository
- Open a new terminal and run `pip install -r requirements.txt`
- Then, run the `main.py` file or manually enter `python main.py` in the terminal

DESCRIPTION:
- A logistic regression model that uses data from the NFL statistics database and current game data to make accurate predictions on the winners of games

DEPENDENCIES:
- Nflreadpy libary
- NumPy
- Seaborn
- Sklearn
- Pandas
- Matplotlib
